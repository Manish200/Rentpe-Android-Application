
Rentpe Android Application

The motive behind  Rentpe application is pretty simple, to reduce paper work, help proprietor in renter's id authentication. It also calculate's various types of renter's bills (eg- Room Rent, Electricity Bill and etc ).

Following are some features which we have developed in our app -

1 -> Advance security features like Aadhar card verification
2 -> Storing the data of renter in SQLite database
3 -> Showing list of renter's information
4 -> Calculation of Bills
5 -> Mobile Notification
